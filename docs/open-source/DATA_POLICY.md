# Data Policy

This repository is open-source and must contain only synthetic, non-sensitive development data.

## Prohibited Content

Do not commit any of the following:

- real contact data (names, personal emails, phone numbers, home/work addresses)
- real donor data
- real partner or vendor data
- real solicitation history
- production database exports or backups
- production reports, letters, or correspondence exports
- personal profile photos or other personal images
- production credentials, tokens, keys, or certificates

## Allowed Content

- synthetic fixtures and examples
- placeholder domains such as `example.org` / `example.com`
- fictional names and addresses used only for testing/demo

## Enforcement

Any accidental commit of prohibited content must be treated as a release blocker and remediated before public release.
